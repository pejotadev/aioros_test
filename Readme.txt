Em nossos jogos com a tecnologia Phygital Touch, os jogadores utilizam seus próprios smartphones e tablets para controlar jogos que estão sendo executados em computadores remotos. Esse controle, implementado como uma página web, estabelece uma conexão em tempo real com nossos servidores.

Implemente uma página web para um jogo do estilo Prize Claw (jogo da garra) constituída por essas views:

* Para esse teste, desconsidere a conexão com o servidor.

View 1: loading (view de entrada)
Exibir o texto "Carregando". Aguarde 3 segundos para exibir a próxima view.

View 2: game
Exibir comandos para mover e descer a garra. Implemente o design fornecido no arquivo Guide.png com as imagens da pasta Images. Os botões de movimento devem ter 2 estados (pressionado e não-pressionado) e tocar o áudio Sounds/button.wav a cada mudança de estado. Ao pressionar o botão para descer a garra, aguarde 3 segundos e avance para a próxima view.

View 3: ranking
Exibir um campo de texto para o jogador informar seu nome de usuário (composto por no máximo 3 letras) e um botão para avançar, que leva à próxima view. Toque o áudio Sounds/button.wav nesse botão também.

View 4: game finished
Exibir o texto "Obrigado por jogar!".

Além disso, faça com que o áudio Sounds/music.mp3 seja executado em todas as views.

Lembre-se que a página será executada em smartphones e tablets utilizando rede móvel.

Caso veja necessidade, sinta-se livre para renomear arquivos, reorganizar pastas, processar arquivos ou realizar qualquer tarefa para que esse projeto seja fácil de ser mantido e disponível ao público com a melhor experiência de usuário possível.

Para finalizar, descreva, em um arquivo de texto, as motivações para cada decisão importante tomada.
